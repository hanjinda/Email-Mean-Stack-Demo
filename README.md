# MEAN Stack Web Demo Framework

## mongodb (500MB free)
- https://mlab.com/
- add the mongodb server address to secrets.js

## Install
1.front-end
- cd frontend/
- npm install
- node app.js (start server)

2.back-end
- cd backend/
- npm install
- node server.js (start server)

## Web View
- localhost:3000 (number you put in frontend/app.js)
